## frames

Publish, browse, and resolve Nebari Frames

### Synopsis

frames is the CLI for the Nebari Frames registry. Use it to publish
Frames authored in your editor, browse Frames your organization has shared,
and print the inheritance-resolved form of a Frame.

### Options

```
      --api-url string            Registry URL; overrides api_url config and FRAMES_API_URL
      --credentials-path string   Credentials file path (for testing)
  -h, --help                      help for frames
  -v, --version                   version for frames
```
